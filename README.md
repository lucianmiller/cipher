# Cipher

#### JavaScript and jQuery practice for Epicodus, 3/10/2021

#### By Lucian Miller

## Description

This software creates a cipher using a sentence the user puts in. This project was use to learn how to link Javascript and jQuery properly, load scripts when the page has fully loaded, and to call and create multiple functions.

## Setup/Installation Requirements

* Clone this repository
* Open file
* Open index.html
* Type a sentence of any length in the prompt that appears
* Refresh page to create a new cipher
